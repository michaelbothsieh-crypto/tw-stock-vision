# 技術指標增強設計文件 (RSI Momentum Design)

## 1. 目標描述
利用 RSI (相對強弱指標) 為目前的 Stock Vision 儀表板注入動態靈魂，透過視覺色彩的變化直觀展現市場熱度，而不增加多餘的圖表視窗，保持極簡專業感。

## 2. 核心設計方案

### 方案一：變色龍線圖 (Chameleon Line)
將 RSI 數值映射至主圖表的價格線顏色，讓使用者一眼看出漲勢是否過熱。
- **指標**: RSI (14)
- **顏色映射邏輯**:
  - `RSI > 70` (超買/強勢): 螢光亮綠色 (#00ffbb) + 外溢發光 (Glow)。
  - `RSI < 30` (超賣): 灰藍色 (#64748b)。
  - `30 <= RSI <= 70` (常態): 翠綠色 (#10b981)。
- **實作技術**: 在 Recharts `AreaChart` 中使用自定義 `linearGradient` 搭配動態計算的 `stops`。

### 方案二：微型動能條 (Mini Momentum Bar)
在資訊側邊欄提供精確的動態量化反饋。
- **位置**: 左側數據面板「技術評級」下方。
- **呈現**: 橫向進度條 (0-100)，顯示具體 RSI 數值與火燄圖示 (當 > 70)。
- **色彩**: 隨數值從冷色 (藍) 漸變至熱色 (紅)。

## 3. 架構與數據流
1. **Frontend (StockChart)**: 在 `useMemo` 中實作 RSI (14) 計算邏輯。
2. **Component Integration**: 
   - `StockChart` 負責繪製變色龍線條。
   - `NeoDashboard` 提取當前最新一筆 RSI 數據並傳遞給 `MomentumBar` 元件。

## 4. 驗證標準
- 選取強勢股時，線圖在價格噴發處應呈現顯眼的螢光綠。
- 切換時間週期時，RSI 應正確重新計算並反映在側邊欄與線圖顏色。
