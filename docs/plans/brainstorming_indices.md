# 技術指標發想紀錄 (Brainstorming: Technical Indicators)

## 專案現況
- **圖表引擎**: Recharts
- **現有功能**: 股價線圖 (Area), 成交量 (Volume, 已保留)
- **需求**: 推薦並選定最適合的技術指標進行實作。

## 決策方案 (Final Selection)

### 方案一：變色龍線圖 (Chameleon Line) [主圖]
- **核心邏輯**: 將 RSI (14) 指標數據與 Close 價格連動。
- **色彩映射**:
    - **RSI > 70 (超買/強勢)**: 線條呈現 `螢光綠 (#00ffbb)` + `外溢發光特效 (glow)`。
    - **RSI < 30 (超賣)**: 線條呈現 `灰藍色 (#64748b)` 或 `暗紅色`。
    - **中間值 (30-70)**: 維持既有的 `翠綠色 (#10b981)`。
- **技術挑戰**: Recharts 的單一 `Area` 組件不支援漸層隨數據點動態變色，需考慮使用「多段線段」或「自定義 SVG 濾鏡」實現。

### 方案二：微型動能條 (Mini Momentum Bar) [資訊欄]
- **位置**: 左側數據面板，技術評級下方。
- **呈現**: 
    - 0-100 的橫向冷熱條。
    - 數值即時顯示 (如 `RSI: 72 🔥`)。
    - 背景色漸層 (冷色到暖色)。
